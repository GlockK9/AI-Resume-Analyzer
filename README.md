# AI Resume Analyzer

An AI-powered resume analysis tool that extracts text from PDFs and evaluates resumes against job descriptions.

## Features
- Upload resume (PDF)
- Extract text using PyPDF
- Analyze skills vs job description
- Simple ATS-style feedback

## Tech Stack
- Python
- Streamlit
- PyPDF

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
